## 推荐的评审视口

- 移动：DevTools 设备模拟 iPhone 12（375 × 812）——巨型主问句在此尺寸最见张力
- 桌面：1440 × 900（容器 640px 居中窄列，两侧大留白）

## 体验路径（核心脊柱）

1. 今日页 → 报头日期戳（06.17 WED）+ 巨型主问句「今天，/提了/么？」三行错位编辑式断句 + 巨型进度数字 0/3 组目标
2. 点底部下划线 CTA「开始一组训练 →」→ 全屏节拍器，准备 3-2-1 → 描边圆环（收紧变朱砂红、放松回 ink）+ 巨型倒计时
3. 完成 → 回今日页，进度 +1，达标时 CTA 红色横扫脉冲 + 报头状态变「DONE · 提了」
4. 进记录页 → 三联大数字栏（连续/总天/总组）+ 月历热力矩阵（4 档：paper/灰/深灰/朱砂红）+ 点格展开日详情
5. 进设置页 → 编号分组（01-05）+ 直角步进器/方拨子开关/分段控件 + 主题即时切换 + 导出 JSON

## 视觉语言（v4 PIVOT）

- **配色**：paper `#FBF7F1` × ink `#141414` × accent `#E63946` 朱砂红，单色强调，零渐变零阴影
- **排版**：超大字号主问句（clamp 64-168px）、巨型 tabular-nums 数字、杂志 eyebrow 小标签（TODAY · 今日 / STREAK / JOURNAL）
- **形态**：直角（圆角 0-2px）、1px 发丝线 + 2-3px 粗线分层、反色 hover、全宽下划线 CTA
- **节拍器**：极简描边 SVG 圆环（反 v3 三层填充），收紧朱砂红描边/放松 ink 描边，中央巨型倒计时

## 主题切换

设置页 → 外观 → 浅 / 深 / 跟随系统；跟随系统模式下，系统切深色会自动跟随。深色模式：背景近黑、文本 paper、accent 微提亮到 `#FF4D5E` 保对比。
